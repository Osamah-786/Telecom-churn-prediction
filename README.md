📌 Overview

This project predicts whether a telecom customer is likely to churn (leave the company) using Machine Learning.
The model analyzes customer data such as tenure, monthly charges, internet service, contract type, payment method, and other features to identify customers at high risk of churning.

The goal of this project is to help telecom companies:

Reduce customer churn
Improve customer retention
Identify high-risk customers early
Increase business revenue

📈 Results

The model successfully predicts customer churn with good accuracy and helps identify important factors affecting customer retention.

Key Insights:

Customers with month-to-month contracts churn more
Higher monthly charges increase churn probability
Long-term customers are less likely to churn
